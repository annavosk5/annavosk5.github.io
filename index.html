HTML & CSS Cheat Sheet

HTML Tags

Tag Name Purpose
<html> Root Element Wraps the entire HTML document.
<head> Head Contains meta information, title, and links to resource links (CSS, fonts, etc.).
<body> Body Contains all content that appears on the page.
<title> Title Sets the browser/window title.
<h1> to <h6> Headers Define headings and subheadings ranging from h1 to h6 (largest bold text to smallest bold text).
<div> Division Groups elements together.
<span> Span Inline container used for styling or grouping within text.
<p> Paragraph Defines a block of text as a paragraph.
<br> Line Break Inserts a single line break in text (has no closing tag).
<ul> Unordered List Creates a bulleted list of items. Used when the order does not matter.
<ol> Ordered List Creates a numbered list of items. Used when the order does matter.
<li> List Item Represents an individual item within an ordered or unordered list.
<img> Image Embeds an image into the page. Requires the src attribute to specify the image file and an optional alt attribute for alternative text.
<video> Video Embeds a video player. Supports attributes of src, controls, autoplay, and loop.
<a> Anchor Creates a hyperlink. Requires the href attribute to specify the destination. Additionally supports the target attribute to customize how the link is opened.
<!-- --> HTML Comment Adds a developer only comment in the code

CSS Properties

Property Purpose Common Values
color Sets the text color. Color names, hex codes, hsl(), and rgb().
background-color Sets the background color of an element. Color names, hex codes, hsl(), rgb(), and transparent.
font-size Controls the size of text. unit values such as px,em,rem, and %.
font-family Defines the typeface used for text Arial, Times New Roman, Serif, Sans-serif, etc.
font-weight Sets thickness/boldness of text. normal, bold, numeric value ranging from 1-1000.
text-align Horizontal alignment of inline text. left, right, center, justify.
margin Adds space outside an elements border. px, em, %, and auto.
padding Adds space inside an elements border. px, em, and %.
border Adds a border around an element. border-width, border-style, border-color.
width Sets the horizontal size of an elemen.t px, %, and auto.
height Sets the vertical size of an element px, %, and auto.
